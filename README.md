## ボルトが通れてエクレアがエトワールで通れない壁の作り方

1. ヒエラルキーからCubeオブジェクトを作る
2. オブジェクトのトランスフォームを変更して作りたい壁の大きさに合わせる
3. オブジェクトのタグとレイヤーを「EclairKeepOut」に変更する
4. メッシュレンダラーのチェックを外す

これで作れます。  
（エトワールで通ろうとしたときの挙動：ボルトにエトワールでぶつかった時の雷のエフェクトが現れず、そのままエクレア人間体になる）
