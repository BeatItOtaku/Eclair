## 操作方法

#### キーボード

移動:		WASD  
ジャンプ:	Space  
ロックオン:	Q  
クイ投げ:	右クリック  
電気流し:	左クリック  
電気移動:	E 

#### ゲームパッド

左スティック:	エクレア移動  
右スティック:	カメラ移動  
右スティック押し込み:	カメラリセット  
R:	ボルト投げ  
L:	電気流す  
ZR:	ロックオン  
ZL:	電気移動
